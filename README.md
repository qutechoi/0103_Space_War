# Space War - 갤러그 스타일 우주선 슈팅 게임

클래식 아케이드 게임 갤러그(Galaga)를 모티브로 한 파이썬 슈팅 게임입니다.

## 게임 소개

우주를 침략하는 적들로부터 지구를 지키세요! 클래식 갤러그 스타일의 슈팅 게임으로, 적 우주선을 격추하고 높은 점수를 획득하세요.

## 주요 기능

### 기본 게임플레이
- **플레이어 우주선**: 좌우 이동 및 총알 발사
- **적 우주선**: 격자 패턴 배치, 자동 이동 및 공격
- **레벨 시스템**: 적을 모두 처치하면 다음 레벨로 진행 (난이도 점진적 증가)
- **생명 시스템**: 3개의 생명으로 시작
- **점수 시스템**: 적 격추 시 점수 획득

### 파워업 시스템 ⭐ (7가지 타입)
| 파워업 | 심볼 | 효과 | 지속시간 |
|--------|------|------|----------|
| **더블샷** | ●● | 2발 동시 발사 | 10초 |
| **트리플샷** | ●●● | 3발 동시 발사 | 10초 |
| **유도탄** | ▲ | 자동 추적 미사일 1발 | 10초 |
| **2발 유도탄** | ▲▲ | 자동 추적 미사일 2발 | 10초 |
| **3발 유도탄** | ▲▲▲ | 자동 추적 미사일 3발 | 10초 |
| **화염방사기** | 🔥 | 초고속 연사 불꽃 (50ms 간격) | 10초 |
| **스마트미사일** | ⭐ | 5대 격추까지 추적하는 특수 미사일 | 5킬까지 |

**특징**:
- 적 격추 시 30% 확률로 파워업 아이템 드롭
- 각 파워업은 고유 색상의 원형 배경과 심볼로 표시
- 더블/트리플샷: 작은 원들로 표시
- 유도탄: 화살표 모양으로 표시
- 화염방사기: 불꽃 모양 심볼
- 스마트 미사일: 별 모양 심볼
- 유도탄은 가장 가까운 적을 자동으로 추적
- 스마트 미사일은 5대를 격추할 때까지 사라지지 않음
- 화염방사기는 짧은 사정거리의 강력한 불꽃 공격

### 비주얼 & 사운드
- **폭발 효과**: 충돌 시 애니메이션 효과
- **파워업 UI**: 실시간 파워업 상태 및 남은 시간 표시
- **사운드 효과**: 총알 발사, 폭발, 적 명중, 파워업 획득, 게임 오버, 레벨업
- **한글 지원**: 시스템 폰트를 사용한 완벽한 한글 표시

## 게임 조작법

| 키 | 동작 |
|---|---|
| **←** | 왼쪽으로 이동 |
| **→** | 오른쪽으로 이동 |
| **SPACE** | 총알 발사 |
| **R** | 게임 재시작 (게임 오버 시) |
| **ESC** | 게임 종료 |

## 게임 규칙

1. **목표**: 모든 적 우주선을 격추하세요
2. **생명**: 총 3개의 생명이 주어집니다
3. **점수**: 적 격추 시 10점 획득
4. **레벨**: 모든 적을 처치하면 다음 레벨로 진행하며, 적의 수가 증가합니다
5. **파워업**: 적 격추 시 30% 확률로 별 모양 아이템이 드롭됩니다
6. **난이도**: 레벨이 올라갈수록 적의 속도가 점진적으로 증가합니다
7. **게임 오버**: 생명이 모두 소진되면 게임이 종료됩니다

- 스마트미사일: 별 모양 (⭐)

## 게임 특징

### 적 AI
- 좌우로 패턴 이동
- 화면 가장자리에 도달 시 방향 전환 및 하강
- 무작위로 총알 발사

### 비주얼 효과
- 폭발 애니메이션
- 배경 별 효과
- 색상으로 구분되는 적 타입

### 사운드 시스템
- **총알 발사**: 레이저 사운드 효과
- **적 명중**: 타격 사운드
- **폭발**: 노이즈 기반 폭발음
- **게임 오버**: 하강 톤
- **레벨업**: 상승 톤
- 프로그래밍 방식 사운드 생성 (외부 파일 불필요)

### 파워업 시스템
- **7가지 다양한 타입**: 일반탄부터 스마트미사일까지
- **심볼 아이템**: 각 파워업을 직관적인 심볼로 표시 (원, 화살표, 불꽃, 별)
- **색상 구분**: 고유 색상의 원형 배경으로 파워업 타입 구분
- **랜덤 드롭**: 적 격추 시 30% 확률로 드롭
- **유도 시스템**: 가장 가까운 적을 자동 추적하는 AI
- **스마트 미사일**: 5대 격추까지 사라지지 않는 특수 무기
- **화염방사기**: 50ms 간격 초고속 연사 (일반 발사 속도의 5배)
- **실시간 UI**: 파워업 상태 및 남은 시간 게이지 바 표시

### 난이도 조절
- 레벨이 올라갈수록 적의 수 증가
- 적의 이동 속도 점진적 증가 (레벨당 +0.1)
- 적의 하강 속도 감소 (더 천천히 내려옴)
- 균형잡힌 게임플레이 경험

## 문제 해결

### pygame이 설치되지 않는 경우

```bash
# Windows
python -m pip install pygame

# macOS/Linux
python3 -m pip install pygame
```

### 게임이 실행되지 않는 경우

1. Python 버전 확인 (3.7 이상 필요)
2. pygame 설치 확인: `pip list | grep pygame`
3. 파일 실행 권한 확인: `chmod +x space_war.py`

### 한글이 표시되지 않는 경우

게임은 자동으로 시스템 폰트를 감지합니다:
- macOS: AppleSDGothicNeo, AppleGothic
- Windows: Malgun Gothic
- Linux: NanumGothic, DejaVu Sans

### 사운드가 들리지 않는 경우

1. 시스템 볼륨 확인
2. pygame.mixer 초기화 확인
3. 사운드 생성 오류 메시지 확인
