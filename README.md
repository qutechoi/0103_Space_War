# 🔮 VibeCoding - 2026년 AI 운세 프로그램

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/)
[![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4-green.svg)](https://openai.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**생년월일과 태어난 시각을 입력받아 AI가 분석하는 2026년 개인 맞춤 운세 프로그램**

## 📖 프로젝트 소개

VibeCoding은 전통적인 사주명리학과 현대 AI 기술을 결합하여 개인 맞춤형 2026년 운세를 제공하는 파이썬 프로그램입니다. OpenAI의 GPT-4를 활용하여 월별 상세 운세와 연간 종합 분석을 제공합니다.

### ✨ 주요 특징

- 🎯 **개인 맞춤 분석**: 생년월일과 태어난 시각 기반의 정확한 개인정보 반영
- 📅 **월별 상세 운세**: 12개월 각각에 대한 세밀한 운세 분석
- 🏆 **8개 운세 영역**: 건강, 금전, 학업/사업, 인간관계, 연애, 가족, 취업/승진, 여행운
- 🤖 **AI 기반 분석**: OpenAI GPT-4를 통한 지능적이고 상세한 운세 해석
- 📊 **종합 평가**: 연간 전체 운세 요약과 핵심 조언
- 💾 **결과 저장**: 완전한 운세 보고서를 텍스트 파일로 저장
- 🌟 **한국어 지원**: 완전한 한국어 인터페이스와 결과 제공

## 🚀 설치 및 설정

### 1. 저장소 클론

```bash
git clone https://github.com/qutechoi/VibeCoding.git
cd VibeCoding
```

### 2. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

### 3. 환경 변수 설정

```bash
# 환경 파일 복사
cp .env.example .env

# .env 파일 편집하여 OpenAI API 키 입력
# OPENAI_API_KEY=your_actual_openai_api_key_here
```

#### OpenAI API 키 발급 방법
1. [OpenAI 플랫폼](https://platform.openai.com/api-keys)에 가입/로그인
2. "Create new secret key" 클릭하여 API 키 생성
3. 생성된 키를 `.env` 파일에 입력

## 📱 사용 방법

### 프로그램 실행

```bash
python fortune_2026.py
```

### 입력 정보

프로그램 실행 후 다음 정보를 입력하세요:

1. **생년월일**: `YYYY-MM-DD` 형식 (예: 1990-05-15)
2. **태어난 시각**: `HH:MM` 형식, 24시간제 (예: 14:30)
3. **성별**: 남/여

### 실행 예시

```
🔮 2026년 운세 프로그램에 오신 것을 환영합니다! 🔮

생년월일을 입력하세요 (YYYY-MM-DD 형식, 예: 1990-05-15): 1995-03-20
태어난 시각을 입력하세요 (HH:MM 형식, 24시간, 예: 14:30): 10:15
성별을 입력하세요 (남/여): 여

입력하신 정보:
생년월일: 1995-03-20
태어난 시각: 10:15:00
성별: 여
2026년 나이: 31세

입력하신 정보가 맞습니까? (y/n): y
```

## 🎯 운세 분석 영역

### 월별 8개 영역 분석

각 달마다 다음 8개 영역에 대해 상세한 분석을 제공합니다:

| 영역 | 분석 내용 |
|------|-----------|
| 🏥 **건강운** | 신체적, 정신적 건강 상태 및 주의사항 |
| 💰 **금전운** | 수입, 지출, 투자, 재정 관리 |
| 📚 **학업/사업운** | 학습능력, 사업성과, 업무효율성 |
| 👥 **인간관계운** | 대인관계, 사회적 네트워킹, 갈등 해결 |
| 💕 **연애운** | 기존 관계 발전, 새로운 만남, 결혼 운 |
| 🏠 **가족운** | 가족 간 화합, 가정사, 효도 운 |
| 💼 **취업/승진운** | 직업 운세, 승진 가능성, 이직 운 |
| ✈️ **여행운** | 국내외 여행, 이사, 환경 변화 |

### 각 영역별 제공 정보

- ⭐ **운세 점수**: ★★★★☆ (5점 만점 별점 평가)
- 🎯 **핵심 키워드**: 해당 월의 주요 테마 3-5개
- 📝 **상세 분석**: 구체적인 운세 해석과 조언 (2-3문단)
- ⚠️ **주의사항**: 피해야 할 행동이나 상황
- 💡 **추천 행동**: 실천하면 좋은 구체적인 행동 지침
